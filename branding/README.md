# VNOffice Branding Assets

Thư mục này chứa các tài nguyên branding cho VNOffice.

## Cấu trúc thư mục

```
branding/
├── logo/
│   ├── vnoffice-logo.png          ← Logo chính (512x512)
│   ├── vnoffice-logo-256.png      ← Logo 256x256
│   ├── vnoffice-logo-128.png      ← Logo 128x128
│   ├── vnoffice-logo-64.png       ← Logo 64x64
│   ├── vnoffice-logo-48.png       ← Logo 48x48
│   ├── vnoffice-logo-32.png       ← Logo 32x32
│   └── vnoffice-logo-16.png       ← Logo 16x16
├── icons/
│   ├── writer.png                 ← Icon cho Writer
│   ├── calc.png                   ← Icon cho Calc
│   ├── impress.png                ← Icon cho Impress
│   ├── draw.png                   ← Icon cho Draw
│   ├── base.png                   ← Icon cho Base
│   └── math.png                   ← Icon cho Math
├── splash/
│   └── splash.png                 ← Splash screen (khi mở app)
└── about/
    └── about.png                  ← Background cho About dialog
```

## Lưu ý
- Logo sẽ được script `apply-branding.sh` tự động copy vào đúng vị trí
- Cần chuẩn bị ảnh PNG với nền trong suốt (transparent)
- Splash screen nên có kích thước 661x169 pixels
